# 🛒 피로마켓 ERD

상품 정보 공유를 위한 커뮤니티 게시판과 중고 물품 판매·구매 기능을 제공하는 피로마켓의 데이터베이스 ERD입니다.

## 🔗 ERD Cloud

[피로마켓 ERD 보러 가기](https://www.erdcloud.com/d/5c77wfejsQXrx8zxp)

> ERD Cloud 공유 설정: **Public**

## 📌 구현 기능

- 회원가입 및 사용자 정보 관리
- 도·시·구 선택형 주소와 상세 주소 관리
- 커뮤니티 게시글 및 게시글 이미지 관리
- 댓글과 대댓글 관리
- 커뮤니티 게시글 좋아요
- 판매 게시글 및 판매 이미지 관리
- 상품 구매와 구매 일자·수량 관리
- 커뮤니티 게시글 해시태그
- 커뮤니티 게시글 유저 태그
- 사용자 간 팔로잉·팔로워 관리
